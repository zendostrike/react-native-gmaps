# react-native-gmaps

A Google Maps native module to show markers

## Installation

```sh
npm install react-native-gmaps
```

## Usage


```js
import { GmapsView } from "react-native-gmaps";

// ...

<GmapsView markers={[]} onMarkerClick={(event) => {}} />
```
